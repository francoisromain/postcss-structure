Grid

This grid system is based on a fixed column width. The breakpoints depends on the number of blocs that can fit into the screen.

Available breakpoints, columns, screen size

$screen_size = ($bloc_width - $gutter) * $col + (2 * $container_padding)

Example for a 240px bloc, 16px gutter and 2 * 16px container padding:

  • xxs: 1 (272px)
  • xs: 2 (512px)
  • s: 3 (752px)
  • m: 4 (992px)
  • l: 5 (1232px)
  • xl: 6 (1472px)
  • xxl: 7 (1712px)
  • xxxl: 8 (1952px)

This is possible to configure the number of blocs for each screensize. For example: m: 3, l: 3.
